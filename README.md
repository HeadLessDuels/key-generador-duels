# Sistema de Generación de Keys con HWID

Sistema web para generar y gestionar keys con vinculación de HWID (Hardware ID).

## Características

- ✅ Generación de keys con expiración personalizable
- ✅ Sistema HWID para vincular keys a dispositivos
- ✅ Reset de HWID para reutilizar keys
- ✅ Interfaz moderna con glassmorphism
- ✅ Fondo animado de bosque oscuro con lluvia
- ✅ Almacenamiento local (localStorage)

## Despliegue en Render.com

1. Sube tu código a GitHub
2. Conecta tu repositorio en Render.com
3. Crea un nuevo "Web Service"
4. Render detectará automáticamente el `package.json`
5. El comando de inicio será: `npm start`

## Instalación Local

```bash
npm install
npm start
```

Abre tu navegador en `http://localhost:3000`

## Tecnologías

- HTML5
- CSS3 (Glassmorphism)
- JavaScript (Vanilla)
- Node.js + Express (para despliegue)
